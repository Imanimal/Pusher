Pusher (Толкатель) - видео игра жанра "Головоломка".
Разрабатывалась в качестве курсового проекта для учёбы в 2016 году.

Заданием было разработать игру "Толкатель" средствами языка Си в консольном приложении.
Персонаж должен уметь перемещаться по уровню и толкать коробки. Ему необходимо достигнуть выхода.

Проект был сдан и завершен в мае 2016 года.
